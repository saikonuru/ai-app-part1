# Ai-app Notes

To install dependencies:

```bash
bun install
```

To run:

```bash
bun run index.ts
```

This project was created using `bun init` in bun v1.2.20. [Bun](https://bun.com) is a fast all-in-one JavaScript runtime.

## Commands

```bash
git init
To add Dev dependency
bun add -d @types/express
bun run dev
```
